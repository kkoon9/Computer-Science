# 1439. 뒤집기

[링크](https://www.acmicpc.net/problem/1439)

|  난이도  | 정답률(\_%) |
| :------: | :---------: |
| Silver V |   52.917    |

## 설계

1. 1의 개수와 0의 개수를 구해준다.
2. 단 연속된 1 또는 0은 한 개로 간주한다. ex) 1111110000일 때, 1의 개수와 0의 개수는 1개이다.
3. 그 중 작은 값을 출력해준다.

### 시간 복잡도

- O(n)으로 처리할 수 있다.
